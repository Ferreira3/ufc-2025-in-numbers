# UFC 2025 Events & Revenue Dashboard (Excel)

A dynamic and interactive **Excel** dashboard for visualizing and analyzing key UFC performance metrics from 2025 data. This project tracks total revenue (Pay-Per-View and gate), top attendance figures, and venue-specific earnings across the U.S. states.
